The course that this website was made for, Web application programming and Hacking (often abbreviated as waph) is a course focused around web development and security, using "hacking" to give us hands on experience with exploiting vulnerabilities, so that we know how to avoid such vulnerabilities. over the course so far, we have learned about html,php, javascript (and relted libraries) as well as css. this project is around the mid point of the course and features making a personal website.
